# Java ATM Management System

## 📌 Description
This is a console-based ATM Management System developed using Core Java.
It supports login authentication, balance inquiry, withdrawal, deposit,
and mini statement functionality.

## 🛠 Technologies Used
- Java
- OOP Concepts
- Collections Framework
- Scanner (User Input)

## ✨ Features
- Secure Login (Account Number & PIN)
- View Balance
- Withdraw Amount
- Deposit Amount
- View Mini Statement
- Exit Option

## ▶️ How to Run
1. Clone the repository
2. Open in IntelliJ / Eclipse
3. Run the main class
4. Enter account number and PIN
